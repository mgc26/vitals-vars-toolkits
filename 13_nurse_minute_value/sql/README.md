# SQL Queries for Nurse Time Valuation

## Planned Queries

1. **nurse_time_allocation.sql** - Extract task-level time data from EHR
2. **nurse_productivity_metrics.sql** - Calculate productivity indicators
3. **patient_outcome_correlation.sql** - Link nurse time to patient outcomes
4. **supply_utilization.sql** - Track supply access and usage patterns
5. **staffing_cost_analysis.sql** - Calculate loaded labor rates and costs
6. **workflow_bottlenecks.sql** - Identify time-consuming processes
7. **roi_tracking.sql** - Measure impact of time-saving interventions

## Database Compatibility

Queries will be designed to work with:
- Epic (Clarity/Caboodle)
- Cerner
- Meditech
- MSSQL-based EMR systems

## Usage Notes

- Queries assume standard EMR data models
- Adjust table/column names based on your system
- Include date range parameters for flexible analysis
- Performance optimized for large datasets
