# Python Analysis Scripts for Nurse Time Valuation

## Planned Scripts

1. **nurse_time_calculator.py** - Calculate cost per minute of RN time
2. **time_allocation_analysis.py** - Analyze time distribution across tasks
3. **roi_model.py** - Model ROI for time-saving interventions
4. **turnover_cost_calculator.py** - Calculate cost of nurse turnover
5. **outcome_correlation.py** - Statistical analysis of time vs. outcomes
6. **scenario_modeling.py** - What-if analysis for workflow changes

## Dependencies

See `requirements.txt` for full list. Key libraries:
- pandas (data manipulation)
- numpy (numerical analysis)
- matplotlib/seaborn (visualization)
- scipy/statsmodels (statistical analysis)

## Setup

```bash
pip install -r requirements.txt
```

## Sample Data

Each script includes sample data generation for testing without real data.
