## Key Metrics

This toolkit helps you measure:
- **Cost per minute** of RN time (loaded labor rate)
- **Time allocation** by task type (clinical vs. administrative)
- **ROI** of workflow interventions
- **Outcome correlation** between bedside time and patient results

## Expected Results

Organizations using these tools typically:
- Identify 30-60 minutes per shift of recoverable nurse time
- Achieve ROI within 6-12 months on targeted interventions
- Reduce nurse turnover through improved workflows
- Improve patient outcomes through increased bedside time
